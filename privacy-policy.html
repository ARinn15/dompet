Kebijakan Privasi

DompetARin

Terakhir diperbarui: 13 Oktober 2025

Selamat datang di DompetARin, aplikasi pencatatan keuangan pribadi. Kebijakan Privasi ini menjelaskan bagaimana kami mengelola data pengguna dan melindungi privasi Anda saat menggunakan aplikasi ini.

1. Informasi yang Kami Simpan

Aplikasi DompetARin tidak mengumpulkan, menyimpan, atau membagikan data pribadi Anda ke server mana pun. Semua data seperti akun, catatan transaksi, dan pengaturan disimpan hanya di perangkat Anda menggunakan localStorage.

  • Kami tidak memiliki akses ke data Anda.
  • Data tidak dikirimkan ke internet atau pihak ketiga mana pun.
  • Data akan hilang jika Anda menghapus aplikasi atau membersihkan penyimpanan lokal.

2. Data yang Anda Masukkan

Aplikasi mungkin meminta Anda untuk membuat akun lokal (username dan password) agar dapat menggunakan fitur pencatatan transaksi. Informasi ini hanya disimpan di perangkat Anda sendiri dan tidak pernah dikirim ke server mana pun.

3. Izin Aplikasi

DompetARin tidak meminta izin apa pun selain yang diperlukan oleh sistem Android untuk penyimpanan lokal. Aplikasi ini tidak mengakses kontak, lokasi, kamera, mikrofon, atau file pribadi Anda.

4. Keamanan Data

Kami berkomitmen menjaga keamanan data Anda. Karena seluruh data disimpan di perangkat Anda, keamanan bergantung pada perangkat dan kebiasaan Anda.

  • Jangan membagikan perangkat tanpa pengamanan layar.
  • Gunakan password yang kuat saat membuat akun.

5. Penghapusan Data

Anda dapat menghapus semua data pribadi kapan pun dengan menghapus aplikasi atau menggunakan fitur Hapus Semua Transaksi di dalam aplikasi. Data Anda akan hilang permanen dan tidak dapat dipulihkan.

6. Perubahan pada Kebijakan Privasi

Kami dapat memperbarui kebijakan privasi ini dari waktu ke waktu. Perubahan akan ditampilkan di halaman ini dengan tanggal pembaruan terbaru.
